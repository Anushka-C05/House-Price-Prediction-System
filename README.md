# House-Price-Prediction-System
Machine Learning project to predict house prices using Linear Regression.

## About the Project

The House Price Prediction System is a Machine Learning project developed using Python. It predicts the price of a house based on features such as area, number of bedrooms, bathrooms, stories, and parking.

The project uses the Linear Regression algorithm to train a model on housing data and estimate house prices for new inputs. It also includes data preprocessing, model training, prediction, and visualization of the results using Matplotlib.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm
- Linear Regression

## Features
- Loads and preprocesses housing data
- Trains a Linear Regression model
- Predicts house prices based on user input
- Evaluates model performance
- Visualizes actual vs predicted house prices
